title: Assets
icon: Boxes
---

Assets are loaded in the following manner:

- `get()` is called by a user on the resource manager.
- The resource manager checks if the resource is already cached/processed.
- If it is cached, the resource document is fetched with its dependencies. Dependencies are resolved recursively so child resources are loaded before their parents.
- If it is not cached, the resource manager invokes resource handlers so they can process the asset when needed.
