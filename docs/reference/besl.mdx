# Byte Engine Shader Language
BESL is Byte Engine's shader language. It exists to make shader code fit naturally into the engine's rendering and material pipeline.

BESL uses a Rust-inspired syntax to make shader programming more ergonomic.

Currently, BESL can only compile down to GLSL.

## Code Samples
### Struct declaration
```rust
Light: struct {
	position: vec3f,
	color: vec3f,
	intensity: f32
}
```

### Function declaration
```rust
mod_by_16: fn (x: i32) -> i32 {
	return x % 16
}
```

### Variable declaration
```rust
x: i32 = 5
```

### Texture intrinsics
```rust
main: fn () -> void {
	let color: vec4f = sample(texture_sampler, uv);
	let texel: vec4f = fetch(texture, coord);
}
```
