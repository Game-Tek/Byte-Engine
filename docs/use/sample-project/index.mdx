Build a small target gallery to learn how Byte Engine systems fit together.

This guide is currently an outline for the complete sample.

The sample is a simple target gallery game where the player can shoot targets to score points.

The finished sample will let you:

- Move the camera with a mouse or gamepad.
- Shoot with the left mouse button or right trigger.
- Increase the score by hitting a target.
- End the session when its timer reaches zero.

Complete the [setup guide](/docs/use/setup) before you create the sample.
