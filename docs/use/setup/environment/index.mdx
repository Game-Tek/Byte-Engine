# Environment setup

Install the common tools on this page, then follow the guide for your operating
system.

## Required

### Install Rust with rustup

Install Rust with `rustup`. The repository pins its nightly toolchain in
`rust-toolchain.toml`, so Cargo downloads and uses the correct version when you
build inside the checkout.

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Set up your platform

Install the SDK, compiler, and packages for your platform before you build a Byte
Engine application.

- [macOS setup](/docs/use/setup/environment/macos)
- [Linux setup](/docs/use/setup/environment/linux)
- [Windows setup](/docs/use/setup/environment/windows)
