Install the native packages and configure AVX2 before you build Byte Engine on
Linux.

## Install system packages

Install the native build, windowing, audio, and Vulkan packages used by Byte's Linux backend.

```bash
sudo apt install -y cmake libwayland-dev libasound2-dev libx11-xcb-dev libxkbcommon-dev libvulkan-dev vulkan-tools vulkan-validationlayers
```

## Enable AVX2 on x64

On x64 Linux, enable AVX2 explicitly instead of using `target-cpu=native`. Add
this configuration to your application's `.cargo/config.toml`:

```toml
[target.x86_64-unknown-linux-gnu]
rustflags = ["-C", "target-feature=+avx2"]
```

## Optional: Install Mold

[Mold](https://github.com/rui314/mold) is an optional linker that can reduce
build times on Linux.

Follow the installation steps in the Mold repository.

## Optional: Install RenderDoc

Install RenderDoc when you need to capture and inspect rendered frames.

```bash
sudo apt install renderdoc
```
