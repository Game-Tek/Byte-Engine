Set up Apple's SDK and the native build tools before you build Byte Engine on
macOS.

## Install the Xcode SDK

Install Apple's command line tools before building on macOS. Byte's macOS backends use the platform SDK frameworks directly, including Metal, AppKit, QuartzCore, Foundation, and Core Audio.

```bash
xcode-select --install
xcrun --show-sdk-path
```

If you installed the full Xcode app, select it so `xcrun`, `clang`, and the Metal
SDK use the same developer directory.

```bash
sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer
sudo xcodebuild -license accept
```

## Install the Metal toolchain

Install the optional Metal compiler component when `xcrun metal` reports that
the Metal Toolchain is missing:

```bash
xcodebuild -downloadComponent MetalToolchain
xcrun -sdk macosx metal --version
```

Keep the selected Xcode developer directory and the downloaded Metal component
from the same Xcode installation.

## Install Homebrew packages

Install the native build tools used by Byte's Rust dependencies.

```bash
brew install cmake pkg-config
```
