---
title: Install Byte Engine
description: How to install the Byte Engine on your system.
icon: Package
---

Use Byte Engine from a source checkout while the public package workflow is in
development.

Add the engine as a path or Git dependency from the checkout that you use for
development.

The future packaged installation command is:

```bash
cargo install byte-engine
```

If Cargo cannot find the package, continue to use the source checkout.
