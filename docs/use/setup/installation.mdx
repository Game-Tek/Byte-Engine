---
title: Install Byte Engine
description: How to install the Byte Engine on your system.
icon: Package
---

Add Byte Engine to your Rust application:

```bash
cargo add byte-engine@0.1.0
```

Byte Engine uses unstable Rust features. Add a `rust-toolchain.toml` file that
selects the engine's supported nightly toolchain:

```toml
[toolchain]
channel = "nightly-2026-05-31"
```

Next, follow the [environment setup guide](/docs/use/setup/environment) for your
platform's graphics, audio, and native build dependencies.

## Use an unreleased revision

To test work that has not reached crates.io, add the Git repository:

```bash
cargo add byte-engine --git https://github.com/Game-Tek/Byte-Engine
```

For engine development, clone the repository and use a path dependency instead.
